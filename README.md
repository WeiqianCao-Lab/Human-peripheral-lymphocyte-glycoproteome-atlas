# Human-peripheral-lymphocyte-glycoproteome-atlas
Prefix	File Name	Description
【1】	ProteinOverlapAnalysis	This R script compares two columns of data across multiple sheets in an Excel file. It calculates overlaps, unique elements, and overlap percentages for each sheet, then outputs summary statistics and detailed lists into a new workbook.

【2】	GlycoSiteStructure	This R script analyzes the secondary structure preference of glycosylation sites. It reads an Excel file containing protein and site information, predicts secondary structures (helix, sheet, coil) of human proteome sequences using the PredictHEC function, then maps glycosylation sites onto these structures. It calculates the distribution and enrichment ratios of glycosylation sites across different secondary structure types compared to both the full proteome and input protein backgrounds, and exports the results to an Excel file.

【3】	GlycoMotifScan	This R script scans protein sequences for NXS/NXT/NXC glycosylation motifs. It reads an Excel file containing protein names and sequences, uses regular expression "N[^P][STC]" to find all motif occurrences (where X can be any amino acid except proline), counts the total matches per protein, records their positions, and exports the results to a new Excel file. The script also provides detailed statistics and validation examples.

【4】	GlycoproteinFreqDist	This R script analyzes protein occurrence frequencies across multiple Excel sheets. It reads each sheet, extracts protein IDs from the first column, counts how many times each protein appears, and categorizes proteins by frequency (1,2,3,4,≥5 times). It generates frequency tables, summary statistics, a stacked bar chart visualization, and exports all results to a new Excel file with separate sheets for each dataset.

【5】	GlycanTypeClassification	This R script classifies N-glycan types from numerical glycan codes. It reads an Excel file containing multiple sheets, parses five-number glycan strings (H, N, A, F, pH) from column D, and classifies each entry into categories including Oligomannose, Complex/Hybrid with various fucosylation/sialylation patterns. It adds classification results as a new column, generates per-sheet statistics, creates an overall summary table, and exports everything to a new Excel file.

【6】	CorrelationHeatmap	This R script generates a correlation heatmap from proteomics data. It reads an Excel file, processes row names and column names, calculates a correlation matrix, and creates a heatmap using pheatmap. The script automatically detects cell types from column names, assigns age group annotations (Young/Aged) based on naming patterns, and customizes axis labels to display cell type names at the second sample position. The resulting heatmap visualizes intercellular correlations with a red color gradient.

【7】	PCA_Glycoprotein	This R script performs PCA analysis on glycoprotein expression data. It reads an Excel file, processes protein expression data, removes constant rows, transposes the matrix, and applies principal component analysis using prcomp(). The script generates a PCA scatter plot with 95% confidence ellipses for each group, calculates variance explained by PC1 and PC2, and saves the plot as a PNG file. Groups are automatically assigned based on sample naming patterns or user-defined categories.

【8】	VolcanoPlot	This R script creates a volcano plot for differential protein expression analysis (Aged vs Young). It reads Excel data containing log2 fold change and p-values, categorizes proteins as up-regulated, down-regulated, or not significant based on thresholds (|Log2FC| > 1, P < 0.05). The plot uses red for up-regulated, blue for down-regulated, gray for non-significant proteins, and highlights a target protein (sp|P04233|HG2A_HUMAN) in yellow with a label. Results are saved as PDF, PNG, and a statistics CSV file.

【9】	GlycanCluster	This R script performs cluster analysis on glycan expression data across six samples (Aged and Young groups, three replicates each). It reads three datasets (Raw, Significance, Normalized) from an Excel file, merges them into a single matrix, and generates a clustered heatmap with row and column clustering enabled. Column annotations distinguish Aged (purple) and Young (blue) groups. A blue-to-red color gradient represents expression levels. 

【10】	PetalPlot_Clock	This R script generates a circular heatmap with an integrated petal plot for visualizing glycoprotein expression data. It processes triplicate measurements for Aged and Young groups across multiple glycosylation sites (130, 136, 270). The inner rings display log2-transformed expression levels with a color gradient, while outer layers indicate replicates. A petal plot (Aged in red, Young in blue) with error bars shows intensity values for three specific sites. The script creates a complex circular visualization combining heatmap sectors, colored arcs for site grouping, and radial labels.
